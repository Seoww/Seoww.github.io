What is a Ceaser Cipher ?

In cryptography, a Caesar cipher, also known as Caesar's cipher, the shift cipher, Caesar's code or Caesar shift, is one of the simplest and most widely known encryption techniques. It is a type of substitution cipher in which each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet. For example, with a left shift of 3, D would be replaced by A, E would become B, and so on. The method is named after Julius Caesar, who used it in his private correspondence.

-taken from Wikipedia page on Ceaser Cipher

How does it work?

Encryption

Put in the number of letters you want to shift forward, as described above, a shift of 3 will result in a ciphertext starting on d, a shift of 5 will result in a ciphertext starting on e and so forth. Type the message you want to encrypt in the textbox below "your text" and click submit, the encrypted text will show below, labeled as encrypted text

Decryption

The letter shift system works the same as the encryption's. However to decipher the encrypted text you have to use the same letter shift that it was encoded with. Again, place the encrypted message in the textbox below "your text" and click submit, the decrypted text will show below, labeled as decrypted text
